# 放映手机冷笑话语料库

说明：
- 这份语料库优先参考 5 个高密度样本：`BV1XieXzMEzT`、`BV1DaZeYwEqh`、`BV1G69BYBEab`、`BV1JQ7pzZE1W`、`BV1bj44z6EPf`。
- 原始材料来自 AI 字幕，存在转写噪声。下面的条目是“按原意做最小还原”的归一化版本，不是逐字转录。
- 语料分成两层：一层是标准问答型冷笑话，一层是放映手机更有辨识度的叙述型冷幽默。

来源：`BV1bj44z6EPf`

2. 说法：地铁站里有好多炸鸡，不愧是美国。
机制：把日常观察直接上升成非常偷懒、但很成立的国别判断。
特征：轻描淡写地给现实贴一个过大的标签。
来源：`BV1bj44z6EPf`

机制：把“装帧”故意滑向“装甲”。

## 三、提炼出来的冷笑话模板

1. 定义式误解
模板：`X 是什么？X 就是更会 X 的 X。`
例子：自学就是学怎样自学。

2. 同音硬拐弯
模板：`A 为什么 B？因为 C 听起来像 D。`
例子：海洋馆很多余、free temple / free tempo。

3. 成语字面化
模板：`把一句抽象成语，当成具体动作处理。`
例子：自己动手，丰衣足食。

4. 专业词汇挪用
模板：`把音乐术语、影视设定、二次元梗拿去解释生活问题。`
例子：音源/姻缘、不上镜、含沙射影。

5. 宏大叙事降格
模板：`本来很重要的事，突然用特别小、特别俗的标准解释。`
例子：大师课学不到东西，但没收门票就不亏。

6. 日常观察上纲上线
模板：`看到一个日常细节，立刻给出特别大的世界判断。`
例子：地铁站里好多炸鸡，不愧是美国。

7. 灾难正向解读
模板：`遇到坏事，先从另一个诡异角度说“太好了”。`
例子：捕鲸船终于捕到鲸了。

8. 拟人化关怀
模板：`把物件、概念、乐器都当作有感受的角色。`
例子：吉他掉水里也可以照顾好自己。

9. 反鸡汤
模板：`本来应该热血励志的位置，反而给一个摆烂式宽慰。`
例子：人活着不是为了努力的，差不多就行了。

10. 一本正经地荒诞
模板：`句子本身越认真，内容越离谱。`
例子：独角鲸很实用，可以做魔杖或者掏耳朵。

## 四、使用建议

- 如果要模仿“放映手机式冷幽默”，不要只学谐音梗。
- 更重要的是她的说法总像是在认真做一件完全不必要的概念解释。
- 她的幽默通常不是热闹型，而是“平静宣布荒唐结论”。
- 她的句子经常很短，前半句像正常陈述，后半句突然侧滑。
- 她很少铺垫笑点，常常直接把包袱当常识说出来。
