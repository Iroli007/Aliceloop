## 1. 色彩的三要素

色彩的三要素是：

1. 色相：颜色的类别，如红、黄、蓝、绿等。
2. 明度：颜色的明暗程度，越接近白色明度越高，越接近黑色明度越低。
3. 饱和度：颜色的纯度或鲜艳程度，饱和度越高越鲜艳，越低越接近灰色。

## 2. 色相环分为几种色

常见的十二色相环可以分为三类：

1. 原色：红、黄、蓝。
2. 间色：橙、绿、紫，由相邻原色混合得到。
3. 复色，也叫第三色：红橙、黄橙、黄绿、蓝绿、蓝紫、红紫，由原色和间色继续混合得到。

十二色相环的顺序通常是：

红、红橙、橙、黄橙、黄、黄绿、绿、蓝绿、蓝、蓝紫、紫、红紫。

## 3. 色彩对比有哪些，如何区分

常见的色彩对比主要有：

1. 色相对比：比较颜色类别是否不同，如红与绿、蓝与橙。
2. 明度对比：比较颜色的深浅差异，如浅黄与深蓝。
3. 饱和度对比：比较颜色的鲜艳程度，如鲜红与灰红。
4. 冷暖对比：比较颜色给人的温度感，如蓝色偏冷、红色偏暖。
5. 补色对比：色相环中相对位置最远的颜色对比最强，如红与绿、黄与紫、蓝与橙。
6. 面积对比：通过颜色所占面积大小形成视觉强调。

区分方法主要看三个维度：

1. 看色相差异，判断是不是不同颜色类别。
2. 看明度差异，判断是偏亮还是偏暗。
3. 看饱和度差异，判断是偏鲜艳还是偏灰。
